# NAME
"Blend"

# Features

あなたがまだ小さい頃、ドリンクバーなどでいろんな飲み物を混ぜてオリジナルドリンクを作った事はありませんか？
その時、奇跡が起きてすごく美味しいドリンクが完成した事もあるかと思います。
このWebサービスは、あの頃の気持ちや楽しさを大人になってもみんなで楽しもうというコンセプトで作られています。

私たちはあなたの独創的なカクテルをお待ちしております。

# Requirement

* ruby '2.5.7'
* rails '5.2.4'
* gem 'devise'
* gem 'bootstrap' '4.2.1'
* gem 'jquery-rails'
* gem 'refile'
* gem 'refile-mini_magick'
* gem 'nested_form_fields'

# Installation

ライブラリはgemfileに記述し、bundle installで導入します。

```bash
bundle install
```

# Usage

* DEMOの実行方法

```bash
git clone https://github.com/blend/~
cd blend
rails s -b 0.0.0.0
```
